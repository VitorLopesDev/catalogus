# Catalogus

Sistema de gerenciamento de acervo pessoal de livros.

## Sobre
O Catalogus permite que usuários cadastrem, organizem e acompanhem 
suas leituras de forma prática e intuitiva.

## Tecnologias
- Java + Spring Boot
- HTML, CSS, JavaScript
- Banco de dados - H2

## Como rodar
 No terminal colocar o comando "mvn spring-boot:run" ou rodar pela própria IDE
